# 認識主控台
認識完程式的輸入輸出後，我們知道 `print` 可以將文字訊息顯示出來，<br/>
顯示的這個地方我們稱為「主控台」。

### 為什麼需要主控台？
主控台中的文字訊息，基本上是不會給一般人看到的，它是屬於工程師也就是你，在設計程式時用來測試用的一個工具，
有的時候為了驗證程式是否正確，我們會想把結果顯示出來觀察看看，但這些訊息又不想給一般使用者看到，就會使用到主控台。

:::caution 錯誤訊息
而當程式有錯誤時，錯誤訊息也會顯示在主控台，幫助我們排除錯誤，當然我們也不希望顯示錯誤訊息給一般人。
:::

著執行以下程式並觀察看看：
<Playground javascript={`prit('哈囉')`} />

你會發現程式並沒有印出「哈囉」，反而顯示了紅色的文字「prit is not defined」，
看到紅色的文字表示程式出現了錯誤，它的意思是 prit 並不是可以執行的方法，這表示方法可能拼錯了。

試著修正錯誤並執行看看吧！
