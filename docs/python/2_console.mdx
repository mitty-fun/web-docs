# 認識主控台
認識完程式的輸入輸出後，我們知道 `print` 可以將文字訊息顯示出來。<br/>
顯示的這個地方我們稱為「主控台」。

### 為什麼需要主控台？
主控台中的文字訊息，基本上是不會給一般人看到的，它是屬於工程師也就是你，在設計程式時測試使用，
有的時候為了驗證寫的程式是否正確，我們會使用主控台把結果印出來觀察看看，

另外當程式有錯誤時，錯誤訊息也會顯示在主控台中，幫助我們來排除錯誤，
也因為這樣所以一般人是不會看到主控台的。

執行看看以下的程式：
<Playground javascript={`prit('哈囉')`} />

你會發現程式並沒有印出「哈囉」反而顯示了紅色的文字「prit is not defined」，
看到紅色的文字表示程式出現了錯誤，它的意思是 prit 並沒有被定義，這表示可能指令英文拼錯了。

試著修正錯誤並執行看看吧！
