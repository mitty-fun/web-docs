# 認識字典
在 Python 程式中，字典是一種被用來**組織**和**收納**資料的方式。

## 字典長什麼樣子
字典和陣列非常相似，我們一樣可以把字典想像成是一個櫃子，這個櫃子有很多個格子可以讓我們存放東西，
每個格子上都會掛上一個名牌，名牌是獨一無二不能重複的，透過名牌我們就可以去存取其對應格子裡的資料。

想像一個情境，我們程式想要去紀錄使用者的健康資料，我們要儲存該使用者的姓名、體重、身高，
因此我們可以用三個變數來儲存：
<Playground python={`
name = '小智'
weight = 52
height = 165
`} />

如果這個程式可以紀錄多個使用者的資料那我們必須改寫程式：

<Playground python={`
s1_name = '小智'
s1_weight = 52
s1_height = 165\n
s2_name = '小美'
s2_weight = 49
s2_height = 161
`} />

我們可以應用字典來將相關聯的資料組織在一起，字典就是多個變數合併在一起
<Playground python={`
s1 = { "name": '小智', "weight": 52, "height": 165 }
s2 = { "name": '小美', "weight": 49, "height": 161 }
`} />

當字典資料很多時，我們可以透過換行方式來呈現，其執行起來是一樣的
<Playground python={`
s1 = {
  "name": '小智',
  "weight": 52,
  "height": 165
}
s2 = {
  "name": '小美',
  "weight": 49,
  "height": 161
}
`} />

## 字典的語法
在 Python 程式語言中，字典用一對大括號 `{}` 表示，要存放的資料寫在大括號中。<br/>
每一筆資料都會用逗號 `,` 隔開，其資料名稱和內容會用冒號 `:` 相隔。

這是一個擁有姓名和年齡兩筆資料的字典，試著把 `user` 印出來看看！
<Playground python={`
user = { name: '托尼', age: 18 }
`}/>

字典用一對大括號表示，冒號的左邊是鍵 key，冒號的右邊是值 value

## 讀取資料
如果你把 `user` 印出來你會看到整個字典完整的被顯示，但如果只是想取得姓名或年齡，
我們可以使用一對中括號 `[]` 來讀取指定的資料。

例如我們想印出年齡可以這麼做：
<Playground python={`
user = { "name": '托尼', "age": 18 } \n
print(user['age'])
`}/>

我們可以傳入該資料的名稱，記得 age 要用字串的方式來表示，不然電腦會把它當作變數導致程式錯誤！

你發現了嗎，讀取字典的資料和讀取陣列一樣，都是使用到一對中括號 `[]` <br/>
只是陣列是使用資料的編號，而字典是使用名稱。

## 新增、修改、刪除

### 修改資料
字典就像是多個變數的集合體，我們可以使用等號 `=` 來設定資料也可以用來改變字典裡的資料。

<Playground python={`
user = { "name": '托尼', "age": 18 }
user.name = '子嘉'
print(user)
`} />

### 新增資料
如果我們設定一個字典不存在的屬性名稱一個資料，程式會出錯嗎？試著執行看看
<Playground python={`
user = { "name": '托尼', "age": 18 }
user["nickname"] = '鋼鐵人'
print(user)
`} />

我們發現程式並不會出錯，而且字典會自動新增對應的資料上去，也就是說我們可以透過這種方式來掛載新的資料到任意的字典上。

### 刪除資料
要刪除字典的屬性可以使用 `del` 語法，試著執行看看：
<Playground python={`
user = { "name": '托尼', "age": 18 }
del user["age"]
print(user)
`} />
