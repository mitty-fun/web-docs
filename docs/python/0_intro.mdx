# Python
Python 程式語言是全世界最火紅的程式語言之一，
根據 [stackoverflow](https://insights.stackoverflow.com/survey/2021) 每年的統計調查，
Python 的熱門以及使用程度在所有的程式語言中排列前三。

JavaScript 之所以會這麼火紅，除了它的語法簡潔優美，受到許多人喜愛外，它擁有非常龐大的社群，
提供各式各樣的工具和框架，你可以選擇用 python 去設計網站後端，也可以用它去做機器學習、資料分析等。

Python 是一門非常值得學習的程式語言，幾乎所有你想透過電腦做到的事情，都可以透過它去完成。
我們接著就馬上來一探 Python 究竟吧！
