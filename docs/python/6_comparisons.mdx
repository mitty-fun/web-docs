# 比較運算
上個章節我們學習了三種基本的資料型態，分別為數值、字串、布林值，<br/>
而說到布林值只有兩種值 `True` 和 `False` 是程式用來表達某件事情的真假。<br/>

程式經常要判斷各種情況，並做出對應的處理，而產生的結果就會是布林值資料，例如比較兩個數字的大小：

<Playground python={`
print(100 > 60)
`}/>

執行以上程式後，會看到主控台印出 `True` 表示比較後的結果為**真**。

接下來我們稍微修改一下程式，<br/>
我們想要設計一個程式，判斷輸入的分數是否及格，如果滿 60 分及格就顯示 `True` 否則 `False`
<Playground python={`
score = prompt('輸入成績？');
print(score > 59);
`}/>

## 布林值資料是什麼
程式透過布林值來表示某個陳述句是否成立。

|布林值|中文|可以用來描述...|
|-|-|-|
|`True`|真|真的、正確、成立、對、是、沒錯、可以|
|`False`|假|假的、不正確、不成立、不對、否、有錯、不行|

<Quiz
  description={`請問底下哪一個句子不能用布林值描述其句子真假？`}
  options={[
    '今天的天氣是晴天。',
    '現在幾點幾分？',
    '我愛妳！',
  ]}
  answer={1}
/>

程式中的布林值用來表示某個描述是否是真的，該描述通常是一個陳述句，
但如果這個句子不是陳述句而是一個「誰、什麼、怎麼、哪裡」的問句，是無法用真或假的布林值來描述。

## 六種比較運算
大於、小於、大於等於、小於等於、相等、不相等

|符號|功能|範例|結果|
|-|-|-|-|
|大於|>|10 > 5|True|
|小於|<|10 < 5|False|
|等於|==|10 == 10|True|
|不等於|!=|10 != 10|False|
|大於等於|>=|15 >= 15|True|
|小於等於|<=|15 <= 15|True|

<Playground python={`
print(10 > 5)
print(10 < 5)
print(10 == 10)
print(10 != 10)
print(15 >= 15)
print(15 <= 15)
`}/>

:::caution 比較相等
比較相等會是兩個等號，因為一個等號已經有「賦值」的作用，為了區別所以改用兩個等號表示。
:::

## 比較相等
字串也可以被用來必較，如果是相等的比較，就是兩邊的字串必須完全一樣，大小寫也必須一樣。
<Playground python={`
print('apple' == 'apple')
print('Apple' == 'apple')
`}/>

:::info 重點整理
* 比較運算的結果會以布林值 `True` 或 `False` 來呈現
* 為了區別一個等號「賦值」的作用，兩個等號才是「比較是否相等」的意思
* 字串比較相等必須字母大小寫都一致才算一樣
:::
