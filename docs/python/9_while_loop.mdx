# while 迴圈

## 什麼是迴圈？
迴圈就是重複不斷執行某個區塊的程式。
迴圈和條件判斷非常相似，都是條件成立就會執行區塊的程式。
迴圈條件成立時就會不斷執行裡面的程式，直到條件不成立才會結束。

## 無限迴圈
底下是一個迴圈，因為 0 < 10 的條件會程式，所以程式會不斷印出哈囉，而且永遠不會停下來，<br/>
因此這種程式又被稱為無窮迴圈，它會導致後面的程式無法執行，千萬不要輕易嘗試。
```python
i = 0
while i < 10:
    print('哈囉！') 
```

## 有限迴圈
為了避免迴圈停不下來，因此會特別設計一個計數器，用來限制迴圈執行的次數。
以下的迴圈程式會執行 10 次。
<Playground python={`
i = 0
while i < 10:
    print('哈囉！') 
    i = i + 1
`}/>

再來看一個例子，
底下的程式會印出數字 0, 1, 2, 3 … 直到印到 99
<Playground python={`
i = 0
while i < 100:
    i = i + 1
    print(i)
`}/>

再來看一個例子，
迴圈每次執行時變數 i 都會加上 2 ，程式最終會印出 1, 3, 5, 7, 9
範例 6-3
<Playground python={`
i = 1
while i < 10:
    print(i)
    i = i + 2
`}/>

:::caution 重點整理
* while 迴圈和 if 非常相似，條件成立時都會執行冒號後的程式
* if 條件是只會執行一次，而 while 迴圈則在條件成立時不斷執行
:::
