# 認識串列
串列是程式用來存放大量資料的一種方式。

## 為什麼需要串列？
想像一個情境，我們想要設計一個程式，用來紀錄某間餐廳，每一位用完餐的客人給予的評價星星，
接著程式可以根據所有的評價資料，統計出平均星星數，以及給予不同星星數的人數。

還記得在程式中我們要紀錄資料，就要透過變數，如果餐廳有 10 位客人給出評價，那我們就要使用 10 個變數去紀錄，
就像下方的程式範例一樣，我們要創造 a, b, c, ... j 總共 10 個變數來儲存。
<Playground python={`
a = 5
b = 4
c = 5
d = 3
e = 4
f = 5
g = 2
h = 5
i = 4
j = 1
`}/>
但我們的資料總不可能永遠固定是 10 個，隨著時間會不斷收集客人的回饋，得到的星星評價資料也會越來越多，
可能會有 100 筆、1000 筆甚至上萬筆資料，所以我們不可能在程式中去創造那麼多的變數，這個時候就要用到串列。

## 串列長什麼樣子？
我們可以把串列想像成是一個櫃子，這個櫃子有很多個格子讓我們存放東西，每個格子上都有一個編號。
在程式中，我們會用一對中括號 `[]` 來表示串列，延續上個範例，如果我們要把 10 個變數的評價資料改用串列存放，可以改寫成這樣：
<Playground python={`
stars = [5, 4, 5, 3, 4, 5, 2 ,5, 4, 1]
`}/>
我們可以把每一筆資料填入中括號中，而每一筆資料之間必須有逗號 `,` 相隔。
這樣是不是整潔多了，我們只要用一個變數去存放一個串列，就可以把所有的資料都記下來。

## 讀取資料
前面說到，串列就像是一個櫃子，它有很多個格子可以存放東西，而每一個格子都有一個獨一無二的編號。
串列中的資料編號，會從編號從 `0` 開始，依序排下去。
透過一對中括號 `[]`，並填入數值就可以去取得串列中指定位置的資料，例如：
<Playground python={`
stars = [5, 4, 5, 3, 4, 5, 2 ,5, 4, 1]\n
print(stars[0]) # 5
print(stars[1]) # 4
print(stars[2]) # 5
`}/>

### 再來一個例子
以下是一個串列存有 5 個水果名稱
<Playground python={`
fruit = ['橘子', '蘋果', '香蕉', '芭樂', '草莓']\n
print(fruit[0]) #橘子
print(fruit[1]) #蘋果
print(fruit[4]) #草莓
`}/>

### 讀取超過範圍的編號
如果讀取串列資料時，傳入的編號超過範圍會不會導致程式錯誤呢？試著執行看看吧：
<Playground python={`
fruit = ['橘子', '蘋果', '香蕉', '芭樂', '草莓']\n
print(fruit[5])
`}/>
執行後你會看到當印出編號 5 的元素時，會出現「IndexError: list index out of range」錯誤訊息，<br/>
IndexError 表示索引編號錯誤。

## 新增、修改、刪除
我們把 Javascript 中的串列比喻成「櫃子」，而這個櫃子更是一種可以靈活組裝的「組合櫃」，<br/>
可以隨時新增格子來裝更多的東西，或拆除格子順便刪掉不要的東西。

### 修改資料
和變數一樣我們可以透過等號 `=` 將資料存入指定的位置，如果裡面已經有資料就會被覆蓋過去。
<Playground python={`
fruit = ['橘子', '蘋果', '香蕉', '芭樂', '草莓']\n
fruit[2] = '芭樂'\n
print(fruit) # ['橘子', '蘋果', '芭樂', '芭樂', '草莓']
print(fruit[2]) #芭樂
`}/>

### 新增資料
append 有新增的意思，可以使用 `append` 語法將資料新增在串列最後。
<Playground python={`
fruit = ['橘子', '蘋果', '香蕉', '芭樂', '草莓']
fruit.append('鳳梨')
fruit.append('芒果')
print(fruit) #['橘子', '蘋果', '芭樂', '芭樂', '草莓', '鳳梨', '芒果']
`}/>

### 移除資料
pop 有彈出的意思，可以使用 `pop` 語法將資料從串列中移除。
<Playground python={`
fruit = ['橘子', '蘋果', '芭樂', '芭樂', '草莓']\n
fruit.pop() #移除最後的元素 - 草莓\n
print(fruit)
`}/>

## 計算長度
可以直接使用 `len` 方法，來取得串列中目前存有多少資料。

<Playground python={`
fruit = ['橘子', '蘋果', '芭樂', '芭樂', '草莓']\n
print(len(fruit)) # 5
`}/>

字串和串列一樣都有 length 屬性表示其內容長度，如果是字串則表示有多少個字。
<Playground python={`
brand = 'orangeapple'
print(len(brand)) # 11
`}/>

## 串列應用
在了解串列的基本概念後，我們來設計一個抽籤程式，這個程式可以輸出多個姓名存在串列中，
並存中抽取出一個幸運兒作為得獎者。

<Playground python={`
from random import randrange\n
names = []
count = int(input('多少個名字?'))\n
for i in range(count):
  names.append(input('請輸入姓名：'))\n
rand = randrange(count)
winner = names[rand]
print('得獎的是 ' + winner)
`}/>
