# 比較運算
在資料型態章節我們介紹了三種原始資料型態，分別為數值、字串、布林值。
我們說布林值型態的資料，是程式用來表達某件事情的真假。

我們在設計程式時，經常會遇到比較的情境，例如：

<Playground python={`
print(10 > 5) # 印出 true
`}/>

## 六種常用的比較符號
大於、小於、大於等於、小於等於、相等、不相等

|符號|功能|範例|結果|
|-|-|-|-|
|大於|>|10 > 5|true|
|小於|<|10 < 5|false|
|等於|==|10 == 10|true|
|不等於|!=|10 != 10|false|
|大於等於|>=|15 >= 15|true|
|小於等於|<=|15 <= 15|true|

<Playground python={`
print(10 > 5)
print(10 < 5)
print(10 == 10)
print(10 != 10)
print(15 >= 15)
print(15 <= 15)
`}/>

## 比較相等
字串也可以被用來必較，如果是相等的比較，就是兩邊的字串必須完全一樣，大小寫也必須一樣。
<Playground python={`
print('apple' == 'apple') #true
print('Apple' == 'apple') #false
`}/>
