# 資料型態
電腦中的資料可以是一段文字、一張圖片、一隻音樂，不同類型的資料，<br/>
程式在執行時會用不同的方式去儲存，區分資料的型態程式才能正確操作這些資料。<br/>
在相同的運算操作下，會因為不同型態的資料，其結果就可能不一樣。

<Playground python={`
print(10 + 20)
print('orange' + 'apple')
`} />

在程式中執行相加運算 `a + b` 將兩個資料加在一起，<br/>
如果 a 和 b 都是數值，就會去做數學相加的計算，如果都是字串，就會將兩段文字合併。

## 三種基本資料型態
Python 程式有設計許多種資料型態，我們先來認識最常見的三種，分別是：數值、字串、布林值。

|型態|英文簡寫|例子|描述|
|-|-|-|-|
|數值|int|1、2、10、15.4、-4、0|由阿拉伯數字組合而成，可以有正負符號、小數點|
|字串|str|'妳好'、"你好"|一定是一對雙引號或單引號包住|
|布林值|bool|True、False|只有兩種值 True 和 False，比較運算結果是用布林值表示|

我們可以使用 `type` 方法來顯示資料的型態
<Playground python={`
print(type(1))
print(type('hi'))
print(type(True))
`}/>

## 為什麼要區分資料型態
因為程式要能分別資料的型態，才能給予對應的操作。<br/>
資料型態在程式中是一個非常重要的概念，在相同的操作下，會因為不同型態的資料，結果就可能不一樣。

例如使用「加號」時電腦會因為等號兩邊的資料型態不同，而得出不同的結果。程式中執行 `a + b` 將兩個對象相加在一起，<br/>
如果 a 和 b 都是數值，就會去做數學相加的計算，如果都是字串，就會將兩段文字合併。
<Playground python={`
print(10 + 20) # 顯示 30 做數學計算
print('orange' + 'apple') # 顯示 orangeapple 做字串相接
`} />

例如使用「乘號」時電腦會因為等號兩邊的資料型態不同，而得出不同的結果。程式中執行 `a * b` 將兩個對象相乘在一起，<br/>
如果 a 和 b 都是數值，就會去做數學相加的計算，如果其中一個是字串，就會字串複製並合併。
<Playground python={`
print(10 * 10) # 顯示 100 數學計算
print('Hi' * 10) # 顯示 HiHiHiHiHiHiHiHiHiHi 字串複製
`} />

## 加法計算
想想看執行以下程式如果分別輸入 10, 20 最後會顯示什麼？
<Playground python={`
a = input('加數：')
b = input('被加數：')
print(a + b)
`}/>

答案是 `1020` 為什麼呢？

因為 a, b 變數的資料型態都是字串，加號 `+` 會把兩邊的字串連起來，得出來的結果就是字串 1020。

:::caution 取得輸入的資料型態
`input` 方法取得使用者輸入的資料會是字串型態，因為程式沒辦法限制使用者輸入什麼樣的內容，<br/>
可以是阿拉伯數字、英文字母、中文甚至各種符號，因此只好統一作為字串讓程式來後續處理。
:::

為了讓以上的程式可以做數學相加計算，我們可以使用 int 方法，將輸入的內容轉換成數字型態，試著執行看看吧：
<Playground python={`
a = input('加數：')
b = input('被加數：')
a = int(a)
b = int(b)
print(a + b)
`}/>

這樣仍無法避免使用者輸入非數字的資料，
如果你試著輸入一些無法轉換成數值的內容，程式就會出錯並出現 `ValueError` 錯誤訊息。

## 一些實驗
我們知道了數字和數字相加，程式會做數學加法計算，字串和字串相加，程式會將兩段字串做合併，<br/>
那如果是字串和數字相加呢？又或是相減、相乘呢？程式會不會出錯？讓我們來實驗看看

<Playground python={`
print('orange' + 10)
print(10 + 'apple')
`}/>

如果數值和字串相加，會得到另外的錯誤訊息 `TypeError` 表示程式出錯了！

我們經常會使用加號來將一些變數資料塞進文字中，例如：
<Playground python={`
name = input('請輸入姓名')
print('你好' + name + '很高興認識你！')
`}/>

:::info 重點整理
* 字串用一對單引號或雙引號包住表示
* 布林值只有兩種 `True` 或 `False`
* `input` 方法取得使用者的輸入，其資料型態為字串
* 字串與字串相加 `+` 會合併兩個字串，數字與字串相加程式會出錯
* 數值與數值相乘 `*` 會複製字串再合併，字串與字串相乘程式會出錯
:::
