## 17 條件判斷
```python
a = float(input('三角形第一個邊長：'))
b = float(input('三角形第二個邊長：'))
c = float(input('三角形第三個邊長：'))

if a == b or b == c or c == a:
    print('等腰三角形')

elif a == b and b == c:
    print('正三角形')

else:
    print('三邊不等長三角形')
```
<Quiz
  description={`請問在輸入合法的三角形三個邊長情況下，何種三角形程式判斷會有錯誤的結果？`}
  options={[
    '正三角形',
    '等腰三角形',
    '三邊不等長三角形',
    '無錯誤',
  ]}
  answer={0}
  explanation={`輸入正三角形三個一樣邊長的數值，在第一個等腰三角形的條件判斷會成立，因此程式會印出等腰三角形並結束，正三角形會被錯誤判斷成等腰三角形。`}
/>


## 18 條件判斷
```python
a = int(input('a'))
b = int(input('b'))
c = int(input('c'))

if a > b:
    if a > c:
        print('a 最大')
    else:
        print('b 最大')
else:
    if b > c:
        print('b 最大')
    else:
        print('c 最大')
```
<Quiz
  description={`執行以下程式分別輸入 10, 10, 5 請問程式會印出什麼？`}
  options={[
    'a 最大',
    'b 最大',
    'c 最大',
    '什麼都不會印出來',
  ]}
  answer={1}
  explanation={`
  Step1 執行 a > b 判斷不成立，因此會執行 else 裡的程式
  Step2 執行 b > c 判斷成立，因此會執行 if 裡的程式
  Step3 印出「b 最大」的文字
  `}
/>
