# 認識變數
變數是程式用來記住事情的方式，就像是便利貼那樣，你可以把事情寫在上面避免忘記。

試著加入第二行 `print(a)` 程式並執行看看
<Playground python={`a = 10`} />

如果把 `a` 給印出來我們會看到 `10` 而不是 `a`，這表示我們成功的將數字資料存入了變數 `a` 中。

## 等號賦值
程式中的等號跟數學的等號是完全不一樣的東西，<br/>
等號有「賦值」的意思，程式中的等號會把等號右邊的內容，存進左邊的對象（通常是變數），<br/>
例如：a = 10 就是把資料數字 10 存進變數 a 裡面。

## 修改變數
如果要修改變數裡面既有的資料，一樣透過等號 `=` 儲存新的內容進去，就可以替換覆蓋既有的內容。<br/>
先觀察以下程式思考看看會印出什麼，接著再去執行它。
<Playground python={`
a = 10 
print(a)
a = 20
print(a)
a = 30
print(a)
`}/>

第一行程式創造一個變數 `a` 並存入數值資料 `10`，下一行會印出 10。<br/>
第二行程式會把變數 `a` 重新存入數值 `20`，原本的資料就被覆蓋，下一行會印出 20。<br/>
第三行程式會把變數 `a` 重新存入數值 `30`，原本的資料就被覆蓋，下一行會印出 30。

如果要修改變數裡的資料，只要使用等號 `=` 重新儲存即可，但原本的資料就會被覆蓋過去並消失！

## 變數的命名限制
* 只能大小寫字母、數字、底線組合而成
* 變數名稱第一個字不能是數字
* 不能跟程式語法衝突，例如：if、while
* 大小寫視為不同變數，例如 Apple 跟 apple 就是不一樣的變數名稱

如果變數名稱是數字開頭，執行時就會出現語法錯誤，既使不執行通常編輯器也會幫你預判出錯誤！<br/>
如果你把底下範例的變數名稱開頭的數字 `1` 移除，程式就能正常了。
<Playground javascript={`
1apple = 10
`} />

:::info 重點整理
* 在 Python 中變數可以用來存放任何型態的資料
* 變數名稱允許英文和數字的組合，但開頭必須是英文字母，且名稱不能和程式語法衝突
* 程式中的等號和一般數學中的等號概念是不同的，會把右邊的東西存入左邊的對象
:::
