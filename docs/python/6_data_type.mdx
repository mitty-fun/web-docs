# 資料資料型態
資料型態在程式中是一個非常重要的概念，在相同的操作下，會因為不同型態的資料，結果就可能不一樣。
例如在程式中執行 `a + b` 將兩個對象相加在一起，如果 a 和 b 都是數值，就會去做數學相加的計算，如果都是字串，就會將兩段文字合併。

<Playground python={`
print(10 + 20) # 顯示 30
print('orange' + 'apple') # 顯示 orangeapple
`} />

在上個章節中我們有介紹到，在程式中要表達一段文字，就要用一段單引號或雙引號包住，而這段文字就是所謂的字串型態資料。

## 三種基本資料型態
JavaScript 程式有設計許多種資料型態，我們先來認識最常見的三種，分別是：數值、字串、布林值。

|型態    |例子|
|-|-|
|數值   |1、2、10、15.4、-4、0|
|字串   |'妳好'、"你好"|
|布林值  |true、false|

數字資料可以被用來做數學運算，字串則是被用來表達一段文字，
布林值則是程式用來表是某件事情的真假，在後面我們會陸續和同學說明。


## 應用加法計算
想想看執行以下程式如果分別輸入 10, 20 最後印出總和變數 c 會是多少呢？

<Playground python={`
let a = prompt('加數：')
let b = prompt('被加數：')
print(a + b)
`}/>

答案是 1020 為什麼呢？

:::caution
要注意的是 prompt 取得使用者輸入的資料一定是「字串」，
所以 a, b 的資料型態都是字串，相加會做字串的串接得出來的結果就是字串 1020。
所以如果要做計算就要型態的轉換。
:::

為了讓以上的程式可以正常運作，我們可以使用 float 將輸入的內容轉換成數字型態。

<Playground python={`
let a = prompt('加數：')
let b = prompt('被加數：')
a = Number(a)
b = Number(b)
print(a + b)
`}/>

## 一些實驗
我們知道數字和數字相加，程式會做數學加法計算，字串和字串相加，程式會將兩段字串做合併。<br/>
那如果是字串和數字相加呢？又或是相減、相乘呢？程式會不會出錯？

試著執行以下範例，你會發現 Python 隱藏了許多神秘的特性！

<Playground python={`
print('orange' + 10)
print(10 + 'apple')
`}/>

<Playground python={`
print(5 * 10)
print('hi' * 10)
`}/>
你有發現什麼特性嗎？字串跟數字相乘竟然會複製字串真是太神奇了！！！

## 字串組合
我們經常會使用加號來將一些變數資料塞進文字中，例如：
<Playground python={`
let name = prompt('請輸入姓名')
print('你好' + name + '很高興認識你！')
`}/>
