# 資料資料型態
整數數字、有小數點的數字、一段文字、一張圖片、一隻音樂，不同類型的資料，<br/>
程式在執行時會用不同的方式去儲存，區分資料的型態程式才能正確操作這些資料。

## 為什麼要區分資料型態
因為程式要能分別資料的型態，才能給予對應的操作。<br/>
資料型態在程式中是一個非常重要的概念，在相同的操作下，會因為不同型態的資料，結果就可能不一樣。

例如使用「加號」時電腦會因為等號兩邊的資料型態不同，而得出不同的結果。程式中執行 `a + b` 將兩個對象相加在一起，<br/>
如果 a 和 b 都是數值，就會去做數學相加的計算，如果都是字串，就會將兩段文字合併。
<Playground python={`
print(10 + 20) # 顯示 30 做數學計算
print('orange' + 'apple') # 顯示 orangeapple 做字串相接
`} />

例如使用「乘號」時電腦會因為等號兩邊的資料型態不同，而得出不同的結果。程式中執行 `a * b` 將兩個對象相乘在一起，<br/>
如果 a 和 b 都是數值，就會去做數學相加的計算，如果其中一個是字串，就會字串複製並合併。
<Playground python={`
print(10 * 10) # 顯示 100 數學計算
print('Hi' * 10) # 顯示 HiHiHiHiHiHiHiHiHiHi 字串複製
`} />

在上個章節中我們有介紹到，在程式中要表達一段文字，就要用一段單引號或雙引號包住，而這段文字就是所謂的字串型態資料。

## 資料分幾種型態？
Python 程式有設計許多種資料型態，常見的基本的資料型態可以分為以下四種：

|型態    |例子|
|-|-|
|整數 int|1、2、10、100、-10、-123、0|
|浮點數 float|0.1、0.25、-0.01、1.0、2.0、-10.0|
|字串 string|"你好"、'hello'、""|
|布林值 boolean|True、False|

## 怎麼區分資料型態
* 整數 vs 浮點數
  * 沒有小數點的就是整數
  * 帶有小數點的一定就是浮點數
  * 注意 1.0 也是浮點數，既使小數點後面是 0。浮點數的資料程式會給予更多的記憶體空間，因為電腦預期你是要做小數運算，所以你寫 1.0 跟 1 電腦在執行時就會判定成不同類型的資料
* 字串
  * 一定是一對雙引號或單引號包住
  * 例如："你好"、'hello'、""
  * 三個 """ 包住的是多行字串
  * 例如："""我是多行字串可以折行"""
* 布林值
  * 只有兩種值 True 和 False
  * 注意開頭一定是大寫
  * 比較運算結果是用布林值表示

## 應用加法計算
想想看執行以下程式如果分別輸入 10, 20 最後印出總和變數 c 會是多少呢？

<Playground python={`
a = input('加數：')
b = input('被加數：')
print(a + b)
`}/>

答案是 1020 為什麼呢？

要注意的是 input 取得使用者輸入的資料一定是「字串」，
所以 a, b 的資料型態都是字串，相加會做字串的串接得出來的結果就是字串 1020。
所以如果要做計算就要型態的轉換。

為了讓以上的程式可以正常運作，我們可以使用 int 將輸入的內容轉換成整數資料型態。

<Playground python={`
a = input('加數：')
b = input('被加數：')
a = int(a)
b = int(b)
print(a + b)
`}/>

字串和字串相加，程式會將兩段字串做合併，那如果是字串和數字相加呢？又或是相減、相乘呢？程式會不會出錯？

<Playground python={`

print(10 + 'apple')
