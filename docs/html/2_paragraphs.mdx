# 文字段落
要在網頁中呈現文字，直接寫上文字就會呈現，<br/>
例如寫上「今天天氣真不錯！」網頁就會呈現相應的文字內容：
<Playground html={`
今天的天氣真不錯！
`} />

但更多時候，我們會將段落的文字，使用 `<p>` 標籤來呈現：
<Playground html={`
<p>今天的天氣真不錯！</p>
`} />

## 為什麼文字要用 p 標籤呈現？
文字不一定要用 p 標籤來呈現，而是根據該文字結果或內容，選擇合適的標籤，例如

如果是一篇文章的段落，我們會選擇 `<p>` 標籤，
如果是條列的事項，我們會選擇 `<ul>` 標籤，
如果是想要粗體醒目，我們會選擇 `<strong>` 標籤，

那如果不加 `<p>` 直接寫上文字呢？
網頁也會顯示你所寫上的文字，但加上標籤可以賦予該內容一個明確的語意角色，
除了幫助網頁搜尋可以更好識別內容外，
對於某些不方便瀏覽網頁的視障者，將內容標記明確可以讓一些視障輔助工具更好的識別網站內容。

將內容明確的標記好，對搜尋引擎友善，通常會獲得更好的分數，讓你的網頁排名更前面更容易曝光！
