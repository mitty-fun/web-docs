# 哈囉 HTML
HTML 是一種標記語言，用來設計網頁。

它的語法簡單易懂，例如我們想要加一個標題到網頁上，我們可以這樣寫：
<Playground html={`<h1>哈囉 HTML</h1>`}/>

在這一系列 HTML 教學中，你將會學到許多設計網頁會用到的元件語法，並附上語法的範例，<br/>
你隨時可以點擊「執行」按鈕直接將網頁跑起來，也可以隨時編輯修改，看看會有什麼樣的變化。

## 什麼是 HTML
HTML 就是用來描述網頁結構與內容的一種語言。
