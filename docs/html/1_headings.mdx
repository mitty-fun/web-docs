# 幫網頁加上標題
不管是網頁、海報、書籍、文宣等等，只要有文字內容通常都會內文以及標題，<br/>
標題通常要醒目而且字體要大，可以讓閱讀的人一眼就知道這個文章的內容主題是什麼。

## 標題大小
網頁一共有 6 種尺寸的標題任君挑選，並用數字 1 ~ 6 來表示，<br/>
h1 是最大的標題，h6 是最小的標題。

<Playground html={`
<h1>我是第 1 大的標題</h1>
<h2>我是第 2 大的標題</h2>
<h3>我是第 3 大的標題</h3>
<h4>我是第 4 大的標題</h4>
<h5>我是第 5 大的標題</h5>
<h6>我是第 6 大的標題</h6>
`} />
執行以上的語法，你會看到標題字體由大至小分別是 h1 至 h6。

## 標題的重要
標題除了在畫面上讓文字呈現大小區別外，他對於搜尋引擎在分析網頁時也非常重要，

:::danger
標題是為了幫助閱讀的人理解內容，千萬不要只是為了放大字體而使用 h 標籤。
:::

如果只是想要讓字體可以放粗體，可以使用 CSS 語法去改變樣式：
<Playground html={`
<h1 style="font-size: 10px;">把最大的標題變超小！</h1>
`} />
CSS 是用來設計網頁外觀樣式，如果想瞭解更多可以之後去閱讀 CSS 系列教學。
