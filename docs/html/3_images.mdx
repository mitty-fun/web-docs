# 加上圖片
幫網站加上圖片可以使用 `<img>` 標籤，這個標籤有一個 `src` 屬性，
裡面要填入圖片的來源網址，這樣網頁才能透過網址去找到圖片並呈現在畫面上。

<Playground html={`
<img src="/static/img/docusaurus.png" />
`} />

試著去搜尋其他的網頁，並在網頁圖片上，使用滑鼠按下右案件，
點選「複製圖片來源」並將複製過來的網址取代 src 屬性裡的網址，
