# 盒子模型
設計網頁就像是在蓋房子砌牆一道牆，網頁上的每一個元素就像是一個個磚塊。<br/>
我們可以把每一個元素看作一個個盒子，可以拆解成四個部分。

* `width`、`height` 內容的長度與寬度
* `padding` 內距，就像是 A4 紙張的影印，印刷的文字不會從紙張的最左上角開始印，<br/>一定會距離紙張上下左右有一定的空隙，這就是 padding
* `border` 邊框，就像是紙盒子的盒子本身厚度
* `margin` 外距，元素與元素之間可以互相推出一個空間，讓他們不要靠得太近
