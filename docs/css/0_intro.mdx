# 哈囉 CSS
CSS 是用來設計網頁 HTML 的外觀樣式的一種語言。

它描述 HTML 元素應該要如何呈現在網頁畫面上。

它的語法簡單易懂，例如我們想要讓一個標題變成紅色，我們可以這樣寫：
<Playground html={`<h1 style="color: red;">哈囉 HTML</h1>`}/>

在這一系列 CSS 教學中，你將會學到許多設計網頁外觀樣式會用到的語法，並附上語法的範例，<br/>
你隨時可以點擊「執行」按鈕直接將網頁跑起來，也可以隨時編輯修改，看看會有什麼樣的變化。

試著修改上面的範例，把 `red` 改成 `blue` 並執行看看會有什麼變化！
