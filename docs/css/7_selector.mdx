# 元素選擇器
你可以撰寫一套規則，去選取你所想要的元素，並去設定樣式。


我們先看到以下的範例，執行後會看到 `<p>` 標籤裡的文字變成了紅色：
<Playground
html={`<p>我只是一個段落</p>`}
css={`p { color: red; }`}
/>

## ID 選擇器
接著我們新增更多的文字段落，看看新增的部分顏色會呈現什麼？
<Playground
html={`
<p>我只是一個段落</p>
<p>我只是一個段落</p>
<p>我只是一個段落</p>
`}
css={`p { color: red; }`}
/>

我們發現包含新增的段落，全部都會呈現紅色。可是如果我們只想指定讓其中一個段落變成紅？

我們可以使用 `id` 幫網頁上的每一個元素賦予一個獨一無二的名字。<br/>
接著修改 css 去選取指定 `id` 的元素來設定樣式。<br/>
在 css 中我們會用 `#` 井字號來表示 id 的名稱。

例如我們想讓中間的段落變成紅色可以這麼做：
<Playground
html={`
<p>我只是一個段落</p>
<p id="red">我只是一個段落</p>
<p>我只是一個段落</p>
`}
css={`
#red { color: red; }`
}
/>

## ID 類別選擇器
接下來我們繼續來做一些變化，如果 5 個段落中想讓其中 3 個變成紅色該怎麼做？<br/>

也許你會想到以下方法：
<Playground
html={`
<p>我只是一個段落</p>
<p id="a">我只是一個段落</p>
<p id="b">我只是一個段落</p>
<p id="c">我只是一個段落</p>
<p>我只是一個段落</p>
`}
css={`
#a { color: red; }
#b { color: red; }
#c { color: red; }
`}
/>

但這樣並不是一個好方法，如果要設定樣式的元素繼續增加，我們就要撰寫更多的 css 去設定。<br/>
而這些設計的樣式只能給指定的元素使用，彈性並不是很高。<br/>
同時這些重複性的樣式設定，如果之後要做修改，就必須全部一起更新。

這時候我們就可以使用類別選擇器。

我們可以使用 `class` 幫網頁上的元素賦予一個類別。<br/>
接著修改 css 去選取指定 `class` 的元素來設定樣式。<br/>
在 css 中我們會用 `.` 點符號來表示 class 的名稱。

<Playground
html={`
<p>我只是一個段落</p>
<p class="red">我只是一個段落</p>
<p class="red">我只是一個段落</p>
<p class="red">我只是一個段落</p>
<p>我只是一個段落</p>
`}
css={`
.red { color: red; }
`}
/>

我們也可以把網頁上的一個個元素，想像成一位位學生，
ID 就像是學生的學號是獨一無二，我們可以透過學號去找到指定的學生，
而 Class 就像是社團，可以很多個學生去參加社團，該學生就會擁有該社團社員的身份。

而且，一位學生是可以參加多個社團，讓我們來看個例子：
<Playground
html={`
<p class="red">我是學生 A</p>
<p class="red bold">我是學生 B</p>
<p class="bold">我是學生 C</p>
`}
css={`
.red {
  color: red;
}
.bold {
  font-weight: 900;
}
`}
/>

在上述的例子中，我們把元素當成三個學生，<br/>
而學生 A 和 B 參加了 red 社團，所以他們變成了紅色，<br/>
而學生 B 和 C 參加了 bold 社團，所以他們變成粗體。

一個元素可以擁有多個類別，每一個類別要用一個空格去分隔。
