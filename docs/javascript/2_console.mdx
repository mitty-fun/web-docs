# 認識主控台
在上一個章節中，我們學會使用 `alert` 方法將文字訊息顯示出來，<br/>
而在 Javascript 程式中還有另外一種顯示文字的方式，就是主控台的文字輸出。

我們可以使用 `console.log` 方法將文字顯示在主控台上，執行看看！

<Playground javascript={`console.log('哈囉');`} />

你會看到黑色的區塊出現了「哈囉」的文字，表示我們成功的將文字顯示在主控台中。

### 為什麼需要主控台？
輸出文字訊息給使用者用 `alert` 方法就可以了，既然這樣為什麼還要多一個 `console.log` 方法的原因，
是因為主控台中的文字訊息，基本上是不會給一般人看到的，它是屬於工程師也就是你，在設計程式時用來測試用的一個工具，
有的時候為了驗證程式是否正確，我們會想把結果顯示出來觀察看看，但這些訊息又不想給一般使用者看到，就會使用到主控台。

:::caution 錯誤訊息
而當程式有錯誤時，錯誤訊息也會顯示在主控台，幫助我們排除錯誤，當然我們也不希望顯示錯誤訊息給一般人。
:::

試著執行以下程式觀察看看：
<Playground javascript={`console.leg('哈囉');`} />

你會發現程式並沒有印出「哈囉」，反而顯示了紅色的文字「console.leg is not a function」。<br/>
看到紅色文字通常表示程式出現了錯誤，而訊息表示 console.leg 並不是可以執行的方法，表示方法名稱可能拼錯了。

試著修正錯誤並執行看看吧！
