# 認識主控台
認識完程式的輸入輸出後，我們知道 `alert` 可以將文字訊息顯示出來。<br/>
而在 Javascript 程式中還有另外一種顯示文字的方式，就是主控台的文字輸出。

我們可以使用 `console.log` 方法將文字顯示在主控台上。

執行看看並觀察：
<Playground javascript={`console.log('哈囉');`} />

點擊 `主控台` 分頁後，你會看到黑色的區塊出現了「哈囉」的文字，
表示我們成功的將文字顯示在主控台中。


### 為什麼需要主控台？
輸出文字訊息給使用者用 `alert` 方法就可以了，既然這樣為什麼還要多一個 `console.log` 的原因，
這是因為主控台中的文字訊息，基本上是不會給一般人看到的，它是屬於工程師也就是你，在設計程式時測試用的輔助工具，
有時候為了驗證寫的程式是否正確，我們會使用主控台把結果印出來觀察看看，

另外當程式有錯誤時，錯誤訊息也會顯示在主控台中，幫助我們來排除錯誤，
也因為這樣所以一般人是不會看到主控台的。

執行看看以下的程式：
<Playground javascript={`console.leg('哈囉');`} />

你會發現程式並沒有印出「哈囉」反而顯示了紅色的文字「console.leg is not a function」，<br/>
看到紅色的文字表示程式出現了錯誤，它的意思是 console.leg 並不是可以執行的方法，這表示指令可能拼錯單詞。

試著修正錯誤並執行看看吧！
