# for 迴圈
`for` 迴圈可以說是 `while` 迴圈的一種語法包裝，在執行指定次數的迴圈時寫起來可以更簡潔。

## 指定次數的迴圈
`while` 迴圈可以做到和 `for` 迴圈一樣的事情，但寫程式時有很多時候經常要用到「指定次數」的迴圈，<br/>
因此有了 `for` 迴圈語法，讓我們撰寫時更簡潔。

在前面我們若要使用 `while` 迴圈去執行指定的次數例如 10 次，我們要這麼寫：
<Playground javascript={`
let i = 0; // 起始值
while (i < 10) { // 結束條件
    console.log(i);
    i += 1; // 變化
}
`}/>

但如果改用 `for` 迴圈就可以將程式改寫成如下：
<Playground javascript={`
for (let i = 0; i < 10; i += 1) {
    console.log(i);
}
`}/>

## 三個要素
在上一個章節 while 迴圈，我們最後做了印數列的練習，並歸納整理出三個要素
* 起始：迴圈開始前會執行一次，通常用來初始化計數迴圈次數的變數
* 條件：條件滿足時迴圈才會不斷執行
* 變化：迴圈每執行完一次，就會做一次變化，通常用來讓計數的變數遞增

而 `for` 迴圈則是將這三個要素，統一寫在小括號中並用分號 `;` 相隔：
<Playground javascript={`
for (起始; 條件; 變化) {
    // 要重複執行的程式
}
`}/>
<Playground javascript={`
for (let i = 0; i < 10; i++) {
    console.log(i);
}
`}/>

* 起始：初始化變數 i 為數值 0
* 條件：當變數 i 數值高於 10 時迴圈就會結束
* 變化：迴圈每執行完一次，變數 i 裡的數值就會增加 1

要注意的是 `for` 迴圈會執行大括號內的程式，執行完才會去執行變化。

:::caution 該使用什麼迴圈
`for` 迴圈能做到的 `while` 迴圈也可以，但 `for` 迴圈更適合在執行**指定次數**的迴圈，<br/>
因為它把迴圈的三個要素都整理在小括號中，所以寫起來更簡單一目了然。<br/>
相反的 `while` 迴圈比較適用於**不確定次數**的迴圈，例如之前的自動打怪程式，怪物的血量可以不固定。
:::


## 數列練習
在前一個 `while` 迴圈的章節中最後，我們做了 5 個印出數列的範例程式，<br/>
接下來我們要用 `for` 迴圈語法去改寫一樣的範例，你也可以回到上個章節去來回比較。

<Playground javascript={`
for (let i = 0; i < 100; i += 1) {
    console.log(i);
}
`}/>
<Playground javascript={`
for (let i = 40; i < 80; i += 1) {
    console.log(i);
}
`}/>
<Playground javascript={`
for (let i = 0; i < 100; i += 2) {
    console.log(i);
}
`}/>
<Playground javascript={`
for (let i = 100; i > 0; i -= 1) {
    console.log(i);
}
`}/>
<Playground javascript={`
for (let i = 50; i > -50; i -= 5) {
    console.log(i);
}
`}/>

:::info 重點整理
* for 迴圈語法小括號 `()` 內的三個元素
  * 初始化，設定負責計數的變數出使數值
  * 條件，當變數數到某個數值時要停止迴圈
  * 變化，迴圈美執行一次數數的變數數值會如何變化
:::
