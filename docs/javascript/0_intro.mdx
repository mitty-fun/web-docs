# 哈囉 JavaScript
JavaScript 程式語言是全世界最火紅的程式語言之一，
根據 [stackoverflow](https://insights.stackoverflow.com/survey/2021#section-most-popular-technologies-programming-scripting-and-markup-languages) 每年的統計調查，
JavaScript 程式語言的熱門以及使用程度，已經蟬聯第一名的寶座非常多年了。

JavaScript 之所以會這麼火紅，是因為它是唯一可以用來開發網頁前端的程式語言，瀏覽網頁是最常見接收資訊的方式，
不管是手機、平板、電腦等裝置都會裝有瀏覽器，所以它的普及度才會如此之高。除了網頁之外，它也可以用來開發手機 APP 或是網站後端等應用程式。
當然它也很適合用來做研究，不管是爬蟲、機器學習、統計工具、資料庫等，都有非常多第三方工具供 JavaScript 程式使用。

JavaScript 非常適合作為入門的程式語言，除了普及度高應用多，它不需要安裝任何環境（只需要有個瀏覽器）就可以開發程式非常方便。
我們接著就馬上來一探 JavaScript 究竟吧！

## 開始之前
此系列教學將會帶你認識 JavaScript 的基礎語法，每一個章節都會有數個範例程式，
這些程式是可以隨時執行並觀察結果，你也可以編輯裡面的程式去實驗。

底下是一個 JavaScript 程式範例，試著將滑鼠移上去，並點擊三角形的按鈕執行看看：
<Playground javascript={`alert('哈囉世界');`} />

執行後會看到網頁跳出了一段文字訊息「哈囉世界」還記得嗎 JavaScript 是可以用來開發網頁的程式，<br/>
所以可以直接在瀏覽器上執行它一點也不奇怪。
