# 認識變數
變數是程式用來記住事情的方式，就像是便利貼那樣，你可以把事情寫在上面避免忘記。

## 創造變數
在 JavaScript 程式中有數種創造變數的方法，這裡我們先來介紹 `let` 語法。
<Playground javascript={`let a;`} />

這樣我們就成功創造一個名為 `a` 的變數了，只是這個變數裡還沒有儲存任何東西。

## 等號賦值
接下來我們會使用到等號 `=` 把資料儲存進這個變數 `a` 當中，程式中的等號跟數學的等號是完全不一樣的東西，

:::caution 等號
程式中的等號有「賦值」的作用，會把等號右邊的內容存進左邊的對象（通常是變數）。
:::

<Playground javascript={`
let a;
a = 10;
`} />

第一行的 `let` 語法是用來告訴電腦要創造一個名為 a 的變數。<br/>
第二行的 `=` 等號語法是告訴電腦，將等號右邊的資料存入左邊的變數 a 中。<br/>
試著加入第三行 `console.log(a);` 程式並執行看看會印出什麼吧。

我們很經常要在創造變數的同時，存入一些預設的資料，因此我們可以將程式合併成一行：
<Playground javascript={`
let a = 10;
console.log(a);
`} />

## 修改變數
如果要修改變數裡面既有的資料，一樣透過等號 `=` 儲存新的內容進去，就可以替換覆蓋既有的內容。<br/>
先觀察以下程式思考看看會印出什麼，接著再去執行它。
<Playground javascript={`
let a = 10;
console.log(a);
a = 20;
console.log(a);
a = 30;
console.log(a);
`} />

## 變數的命名限制
* 只能大小寫字母、數字、底線、中文字
* 變數名稱第一個字不能是數字
* 大小寫視為不同變數，例如 Apple 跟 apple 就是不一樣的變數名稱
* 不能跟程式語法衝突，例如：if、while

如果變數名稱是數字開頭，執行時就會出現語法錯誤，既使不執行通常編輯器也會幫你預判出錯誤！<br/>
如果你把底下範例的變數名稱開頭的數字 `1` 移除，程式就能正常了。
<Playground javascript={`
let 1apple;
`} />

## 挑戰你的大腦
創造兩個變數 `a` 和 `b` 並都分別存入數值 `10` 和 `20`，想想看最後 `a` 和 `b` 是存入什麼樣的內容？<br/>
加入 `console.log(a, b);` 程式到最後一行，並執行看看 `a` 和 `b` 變數裡的數值會是多少？
<Playground javascript={`
let a = 10;
let b = 20;
a = b;
b = a;
`} />

我們再來做一些變化，創造兩個變數 `x` 和 `y` 並都分別存入數值 `10`，<br/>
接著做了兩行的的相加計算，試著想想看最後 `a` 和 `b` 會存入什麼樣的內容？<br/>
<Playground javascript={`
let x = 10;
let y = 10;
x = x + y;
y = x + y;
`} />

:::info 重點整理
* 在 JavaScript 中變數可以用來存放任何型態的資料
* 變數名稱允許英文和數字的組合，但開頭必須是英文字母，且名稱不能和程式語法衝突
* 程式中的等號和一般數學中的等號概念是不同的，會把右邊的東西存入左邊的對象
:::
