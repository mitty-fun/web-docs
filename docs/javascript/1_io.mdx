# 輸入與輸出
電腦是為了幫助人類處理複雜繁複的事情，為了正確的交辦電腦任務，並取得完成後的結果，<br/>
必須有一種方式可以跟電腦溝通，這個方式就是輸入和輸出。

## 什麼是輸入？什麼是輸出？
輸出就是程式傳遞訊息給使用者，它也可以是文字、螢幕畫面、喇叭播放聲音等。<br/>
輸入就是程式接收訊息自使用者，它可以是滑鼠移動、鍵盤輸入、鏡頭的畫面等。

程式的輸出就像是嘴巴，可以**發送**訊息給人類，會使用 `alert` 指令（方法）。<br/>
程式的輸入就像是耳朵，可以**接收**訊息自人類，會使用 `prompt` 指令（方法）。<br/>

大部分的程式語言都會設計**文字**的輸入輸出方式，因為這是最簡單直接的方式！

## 輸出文字
在 JavaScript 程式語言中，可以使用 `alert` 指令讓電腦輸出文字，試著執行看看吧！
<Playground javascript={`alert()`}/>

執行完上面的程式後，後你會看到網頁會跳出一個文字框，裡面除了標題外並沒有其他的文字訊息，<br/>
這是因為我們還沒有明確告訴電腦要說什麼。

:::caution 執行指令
程式除了打上 `alert` 指令外，後面還加上了一對小括號，在 JavaScript 中小括號具有執行的意思。
:::

為了讓電腦可以顯示它想說的話，我們可以把文字加在小括號中，<br/>

例如，我們想讓電腦說出「你好」這段文字，我們可以這麼做：
<Playground javascript={`alert('你好')`}/>

:::caution 表達字串
在程式中表達一段文字，我們會用一對單引號或雙引號包住，
你可以把單引號 `'` 換成雙引號 `"` 並執行看看。
:::

## 輸入文字
在 JavaScript 程式語言中，可以使用 `prompt` 指令讓電腦輸出文字，試著執行看看吧！
<Playground javascript={`prompt()`}/>

執行後會你會看到，網頁跳出了一個輸入框，讓我們輸入一些文字給電腦。

如果只是看到一個輸入框，使用者會不知道自己要輸入什麼樣的訊息給電腦，
所以可以把一些提示文字寫在小括號中（和 alert 指令一樣）。

例如我們想詢問使用者的名稱，可以這樣寫：
<Playground javascript={`prompt('請輸入姓名：')`}/>

## 來設計一個聊天機器人吧
接著我們來設計一個可以做簡單對話的聊天機器人，它會詢問使用者的名稱，接著再根據名稱和對方打招呼。

電腦在執行程式時，會一行一行由上而下逐一執行，所以兩行的程式會先執行第一行接著才會執行第二行。

試著執行看看吧。
<Playground javascript={`
let user = prompt('你的名字是？');
alert('你好' + user + '!');
`} />

這裡出現了一些還沒學過的語法，先不用擔心，我們會再接下來的章節陸續介紹到，<br/>
首先看到第一行程式，它做的事情就是把我們所輸入的文字內容紀錄在 `user` 變數中，<br/>
接著看道第二行程式，它會將「你好」文字和輸入的名字組合起來再顯示出來。

:::caution 關於分號
在 JavaScript 程式中分號 `;` 如同是句號一樣，每當我們執行完一段完整的程式後就會加上。<br/>
也許你會發現不加程式似乎也不會出錯，這是因為在執行前電腦會試著幫你補起，<br/>
但現在我們建議你要加上 `;` 分號，當你已經熟悉背後的原理機制，你可以再自己選擇加與不加。
:::

:::info 重點整理
* 程式中的這些可以執行指令，我們稱呼為 `方法`。
* `alert` 是輸出文字給使用者的方法。
* `prompt` 是取得使用者輸入的方法。
* 程式中的分號 `;` 就像是文章裡的句號，告訴電腦程式執行是一段完整的程式。
:::
