# 輸入與輸出
電腦是為了幫助人類處理複雜繁複的事情，為了正確的交辦電腦任務，並取得電腦完成後的成果，<br/>
必須有一種方式可以跟電腦溝通，這就是輸入和輸出。

## 什麼是輸入？什麼是輸出？
輸出就是程式傳遞訊息給使用者，它也可以是文字、螢幕畫面、喇叭播放聲音等。<br/>
輸入就是程式接收訊息自使用者，它可以是滑鼠移動、鍵盤輸入、鏡頭的畫面等。

程式的輸出就像是嘴巴，可以**發送**訊息給人類，會使用 `alert` 指令（方法）。<br/>
程式的輸入就像是耳朵，可以**接收**訊息自人類，會使用 `prompt` 指令（方法）。<br/>

近乎所有的程式語言都會設計**文字**的輸入輸出方式，因為這是最簡單直接的方式！

## 輸出文字
在 JavaScript 程式語言中，可以使用 `alert` 指令讓電腦輸出文字，試著執行看看吧！
<Playground javascript={`alert()`}/>

你應該注意到，程式除了打上 `alert` 指令外，後面還加上了一對小括號，
在 JavaScript 中小括號具有執行的意思，另外指令在程式中我們又會稱為 `方法`。

執行完上面的程式後，後你會看到網頁會跳出一個文字框，裡面除了標題外並沒有其他的文字訊息，<br/>
這是因為我們還沒有明確告訴電腦要說什麼。

為了讓電腦可以顯示我們它想說的文字，我們可以把文字加在一對小括號中，<br/>
注意！在程式中我們會用一對單引號或雙引號包住文字，來讓電腦知道它是一個字串。

例如，我們想讓電腦說出「你好」這段文字，我們可以這麼做：
<Playground javascript={`alert('你好')`}/>

在程式中表達一段文字，我們會用一對單引號或雙引號包住，<br/>
你可以試著修改裡面的文字並再執行一次，看看結果會不會跟著變化。

## 輸入文字
在 JavaScript 程式語言中，可以使用 `prompt` 指令讓電腦輸出文字，試著執行看看吧！
<Playground javascript={`prompt()`}/>

執行後會你會看到，網頁跳出了一個輸入框，讓我們輸入一些文字給電腦。

如果只是看到一個輸入框，使用者會不知道電腦想要它輸入什麼樣的訊息，
所以可以把一些提示文字寫在小括號中。

例如，我們想詢問使用者的稱呼，可以這樣寫：
<Playground javascript={`prompt('請問你的名字是？')`}/>

## 來設計一個聊天機器人吧
接著我們來設計一個可以做簡單對話的聊天機器人，它會詢問使用者的名稱，接著再根據名稱和對方打招呼。

電腦在執行程式時，會一行一行由上而下逐一執行，所以兩行的程式會先執行第一行接著才會執行第二行。

試著執行看看吧。
<Playground javascript={`let name = prompt('你的名字是？');
alert('你好' + name + '!');`} />

這裡出現了一些沒見過的語法，先不用擔心，我們會再接下來的章節陸續介紹到，<br/>
首先看到第一行程式，它做的事情就是把我們所輸入的文字內容紀錄在 `name` 變數中，<br/>
接著道道第二行程式，它會將「你好」文字和輸入的名字一起顯示出來，在 JavaScript 程式中，相加符號能把字串連接在一起。
