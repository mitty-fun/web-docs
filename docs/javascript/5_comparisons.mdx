# 比較運算
在資料型態章節我們介紹了三種原始資料型態，分別為數值、字串、布林值。<br/>
我們說布林值型態的資料，是程式用來表達某件事情的真假。

我們在設計程式時，經常會遇到比較的情境，例如：
<Playground javascript={`
console.log(10 > 5);
`}/>

執行後會看到主控台印出 `true` 表示比較後的結果為**真**。

## 布林值資料是什麼
程式會透過布林值來表示某個陳述句是否成立<br/>
布林值 `true` 表示真的、正確、成立、對、是。<br/>
布林值 `false` 表示假的、不正確、不成立、不對、否。

<Quiz
  description={`請問底下哪一個句子不能用布林值描述其真假？`}
  options={[
    '今天的天氣是晴天',
    '今天幾月幾號',
    '我愛妳',
  ]}
  answer={1}
/>

程式中的布林值用來表示某個描述是否是真的，該描述通常是一個陳述句，
但如果這個句子不是陳述句而是一個「誰、什麼、怎麼、哪裡」的問句，是無法用真或假的布林值來描述。

## 六種常用的比較符號
大於、小於、大於等於、小於等於、相等、不相等

|符號|功能|範例|結果|
|-|-|-|-|
|大於|>|10 > 5|true|
|小於|<|10 < 5|false|
|等於|==|10 == 10|true|
|不等於|!=|10 != 10|false|
|大於等於|>=|15 >= 15|true|
|小於等於|<=|15 <= 15|true|

<Playground javascript={`
console.log(10 > 5);
console.log(10 < 5);
console.log(10 == 10);
console.log(10 != 10);
console.log(15 >= 15);
console.log(15 <= 15);
`}/>

## 比較相等
字串也可以被用來必較，如果是相等的比較，就是兩邊的字串必須完全一樣，大小寫也必須一樣。
<Playground javascript={`
console.log('apple' == 'apple');
console.log('Apple' == 'apple');
`}/>

:::caution 重點整理
* 為了區別一個等號「賦值」的作用，兩個等號才是「比較是否相等」的意思
* if 條件判斷的條件，最後接收的是一個布林值
:::
